# Snake Water Gun (Python)

A simple command-line game in Python based on the classic Snake–Water–Gun rules.

## Rules
- Snake drinks Water → Snake wins
- Water disables Gun → Water wins
- Gun kills Snake → Gun wins

## How to Play
Enter:
- `s` for Snake
- `w` for Water
- `g` for Gun

## Run Locally
```bash
python Snake-water-gun.py
